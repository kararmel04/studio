# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Lancer le projet en local

### Prérequis

- Node.js (version 18 ou supérieure)
- npm ou yarn

### Installation

1. Clonez ce repository et naviguez dans le dossier du projet :

```bash
cd project
```

2. Installez les dépendances :

```bash
npm install
```

### Démarrage

Lancez le serveur de développement :

```bash
npm run dev
```

L'application sera disponible à l'adresse [http://localhost:9002](http://localhost:9002).

### Autres commandes

- **Build de production** : `npm run build`
- **Démarrer en production** : `npm start`
- **Vérification du linting** : `npm run lint`
- **Vérification des types** : `npm run typecheck`
